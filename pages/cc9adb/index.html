# v1.0.3 beta (2022/11/8)

feat: 配置中心更新

  • 1、新增配置下发用模板
  • 2、新增 TTP 模板页面
  • 3、新增 FSM 模板页面
  • 4、新增配置合规性检查页面

docker exec -it netaxe-server /bin/bash
更新菜单目录
python manage.py init_system_menu

python manage.py makemigrations config_center
python manage.py migrate config_center
