# 👍 资源管理

  • ⚡️ 网络设备

    • 需要先到9999/admin后台新建设备管理账户
    • 新建默认或测试机柜
    • 点击资产录入按钮
    • 填写和勾选字段、设备在线、数据采集是
    • 点击选中设备条目行首的选择框、右上角关联采集
    • 点击选中设备条目行首的箭头，展开行，选择关联账户，选中新建的设备账户
    • 展开行查询账户。输入二级密码:netaxe_second
    • WEBSSH验证登录是否成功
    • 数据excel导入-验证中
    • 数据excel导出可使用

  • ⚡️ 接口清单

    • 设备所有接口的清单信息、需要信息采集任务后生成记录
